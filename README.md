## Project Overview
The Book Library App is a frontend application built with **React** that allows users to search for books and view basic details about them. The app uses the **Open Library API** to fetch real book data based on user searches.

This project is part of a **Frontend Capstone Project** and focuses on API integration, React components, and responsive UI design.

---

## Features
- Search books by title, author, or keyword
- Display a list of books with cover, title, and author
- View basic book details
- Handle empty results and errors
- Responsive design

---

## Tech Stack
- React (Vite)
- JavaScript
- Tailwind CSS
- Open Library API
